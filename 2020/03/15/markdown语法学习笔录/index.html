初识

我刚开始使用Markdown时，我主要掌握的语法为以下几个。

1. 标题：#(可为多个)+空格+内容

  • 示例： # 标题为一级标题
      ## 标题为二级标题
      ……
      ###### 标题为六级

2. 粗体、斜体、斜体加粗、加删除线

  • 示例 **粗体**、*斜体*、***斜体加粗***、~~加删除线~~

    以下为实际效果
    粗体、斜体、斜体加粗、加删除线

3. 换行

  • 连续两个空格以上+回车

4. 首行缩进（空格）

  • 关于缩进有以下几种；
    &emsp;,&nbsp;,&ensp;,&thinsp;,&thinsp;,&zwj;.
    推荐第一种，和中文空格类似。详细看引用。

    markdown编辑器中可以使用的6种空格标记：https://blog.csdn.net/testcs_dn/article/details/78957685
