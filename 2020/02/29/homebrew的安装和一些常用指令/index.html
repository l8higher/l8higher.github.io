homebrew的安装和一些常用指令

系统：macOS 工具：iterm2

何物？

Homebrew是一款软件安装包管理系统，简单来说，用它可以非常方便的安装更新卸载应用。

不会安装？

那好吧，简单说一下，打开终端，输入以下代码，回车，完事儿。
ruby -e “$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)”
ruby命令macOS自带的，不用考虑还用安装。
（因为GitHub嘛，下载肯定会比较慢，所以这个时候给你的终端开个科学上网就很有必要了。所以，建议在有科学上网环境的时候使用这种方式，也建议大家去摸索一下这个环境，这毕竟刚需。）

如何使用？

homebrew的几个核心命令

  • brew install 程序 :主要用于安装命令行程序及其相关环境，如以下brew install Java
  • brew cask install 应用：用于下载安装已经编译好的应用包（图形GUI应用）,如
    `brew cask install google-chrome`
  • brew update:更新brew自身
  • brew search 应用：搜索应用，比如brew search vim
  • brew cleanup清理
  • brew update && brew upgrade && brew cleanup:一键更新清理

对于brew来说更新软件很容易，而用brew cask来更新就要费一番了
安装 brew-cask-upgrade brew tap buo/cask-upgrade
ps:第一次安装upgrade失败了，不过第二次尝试成功。可能是刚换代理的原因。
以下为brew cask更新软件的相关代码：

  • brew cu:更新所有过时应用
  • brew cu 应用:更新指定应用
  • brew upgrade:更新已安装的应用程序

行文至此，关于homebrew的常规操作都在这里了，以后可能会补充更多关于homebrew的底层的一些东西。
